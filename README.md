# Igrok

Terminal-based music player with YouTube download and real-time audio visualization.

## Features
- YouTube video/playlist download
- High-quality audio extraction
- mpv integration
- Real-time Cava visualization

## Installation

### From AUR
```bash
yay -S igrok
```

## Usage
```bash
igrok
igrok --url "https://youtube.com/watch?v=..."
```

## Dependencies
- yt-dlp
- mpv
- cava
